# FXRuby

[github-homepage](https://github.com/larskanis/fxruby)

FXRuby is a library for developing powerful and sophisticated cross-platform graphical user interfaces (GUIs) for your Ruby applications. Itâ€™s based on the FOX Toolkit, a popular open source C++ library developed by Jeroen van der Zijp.



## Installation

on Unix requires FOX development headers and libraries installed:

````shell
sudo apt-get install g++ libxrandr-dev libfox-1.6-dev
````

Then, install the gem:

````shell 
sudo gem install fxruby
````
